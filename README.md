# pongame
pong clone made with pygame

### requirements

- python 3
- pip
- virtualenv


### setup

- clone the repo, create and activate a virtualenv 
  
        git clone https://github.com/samuele-mattiuzzo/pongame.git
        cd pongame
        virtualenv env
        . env/bin/activate

- install the requirements

        pip install -r requirements.txt

- run the game

        python3 pongame.py
